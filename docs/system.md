# System

NewsAware is a structured, knowledge-aware agent for evaluating text.

## Pipeline

Input → Analyze → Check Evidence → Refine

## Components

- **Analyze**: Extract sentiment and identify the main claim  
- **Check Evidence**: Evaluate whether the claim is supported by evidence  
- **Refine**: Adjust credibility and generate reasoning  

## Design Principle

The system prioritizes:
- Evidence-aware reasoning  
- Structured evaluation  
- Minimal assumptions beyond input text
