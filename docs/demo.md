## Example

**Input:**
"Social media causes depression in teenagers, studies show."

**Output:**
- Sentiment: Negative  
- Credibility: Medium–Low  
- Reason:
  - Uses vague attribution ("studies show")  
  - Does not cite specific sources  
  - Overgeneralizes complex relationship
